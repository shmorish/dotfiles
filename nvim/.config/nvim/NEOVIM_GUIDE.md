# Neovim 設定ガイド

## 概要

| 設定 | 値 |
|------|-----|
| Leader キー | `Space` |
| キーシーケンスタイムアウト | 300ms |
| 更新時間 (diagnostics) | 300ms |
| プラグインマネージャー | lazy.nvim |
| カラーテーマ | kanagawa-dragon (透過有効) |

## キーバインド

### モード切り替え・基本操作

| キー | モード | 動作 |
|------|--------|------|
| `jj` | Insert | Normal モードへ戻る (Esc 代替) |
| `Esc Esc` | Normal | 検索ハイライト消去 |
| `Space n` | Normal | 検索ハイライト消去 |
| `rn` | Normal | 相対行番号の切り替え |

### カーソル移動

| キー | モード | 動作 |
|------|--------|------|
| `j` | Normal / Visual | 下へ移動 (折り返し行対応) |
| `k` | Normal / Visual | 上へ移動 (折り返し行対応) |
| `H` | Normal / Visual | 行頭へジャンプ |
| `L` | Normal / Visual | 行末へジャンプ |

### 編集

| キー | モード | 動作 |
|------|--------|------|
| `+` | Normal | 数値インクリメント |
| `-` | Normal | 数値デクリメント |
| `Y` | Normal | 行末までヤンク |
| `p` | Normal | 貼り付け (自動インデント) |
| `P` | Normal | 上に貼り付け (自動インデント) |

### コマンドショートカット

| キー | 動作 |
|------|------|
| `Q` | `:q` - 終了 |
| `W` | `:w` - 保存 |
| `QA` / `Qa` | `:qa` - 全バッファ終了 |
| `WQ` / `Wq` | `:wq` - 保存して終了 |
| `WQA` / `Wqa` | `:wqa` - 全バッファ保存して終了 |

### ウィンドウ操作

| キー | 動作 |
|------|------|
| `ss` | 水平分割 |
| `sv` | 垂直分割 |
| `sh` | 左のウィンドウへ移動 |
| `sj` | 下のウィンドウへ移動 |
| `sk` | 上のウィンドウへ移動 |
| `sl` | 右のウィンドウへ移動 |
| `sH` | ウィンドウを左へ移動 |
| `sJ` | ウィンドウを下へ移動 |
| `sK` | ウィンドウを上へ移動 |
| `sL` | ウィンドウを右へ移動 |
| `s,` | 幅を 3 縮小 |
| `s.` | 幅を 3 拡大 |
| `s-` | 高さを 3 縮小 |
| `s=` | 高さを 3 拡大 |
| `s0` | ウィンドウサイズを均等化 |

### LSP

|------|------|

### 診断リスト (Trouble)

| キー | 動作 |

### ジャンプ移動 (Hop)

| キー | モード | 動作 |
|------|--------|------|
| `f` | Normal | カーソル以降の文字へジャンプ (同一行) |
| `F` | Normal | カーソル以前の文字へジャンプ (同一行) |
| `t` | Normal | カーソル以降の文字の手前へジャンプ (同一行) |
| `T` | Normal | カーソル以前の文字の手前へジャンプ (同一行) |

### 補完 (nvim-cmp)

| キー | 動作 |
|------|------|
| `Ctrl + b` | ドキュメントを上へスクロール |
| `Ctrl + f` | ドキュメントを下へスクロール |
| `Ctrl + Space` | 補完をトリガー |

### ターミナル

| キー | モード | 動作 |
|------|--------|------|
| `Space t t` | Normal | ターミナルを開く |
| `Space t s` | Normal | 下に分割してターミナルを開く |
| `Space t v` | Normal | 垂直分割してターミナルを開く |

## LSP / 補完設定

### LSP セットアップ

- **管理:** `mason.nvim` + `mason-lspconfig.nvim` で自動インストール
- **Capabilities:** すべてのサーバーが `cmp_nvim_lsp` の補完機能を利用

### カスタム設定のある言語

| 言語 | サーバー | 設定 |
|------|---------|------|
| Rust | `rust_analyzer` | `check.command = "clippy"` |
| Go | `gopls` | `unusedparams = true`, `staticcheck = true` |
| その他 | 自動 | デフォルト設定 |

### 補完ソース優先度

| 優先度 | ソース | 説明 |
|--------|--------|------|
| 1 | `nvim_lsp` | LSP による補完 |
| 2 | `lazydev` | Lua API ドキュメント |
| 3 | `copilot` | GitHub Copilot |
| 4 | `buffer` | バッファキーワード (2文字以上) |
| 5 | `path` | ファイルパス |
| 6 | `cmdline` | コマンドライン |

### 補完メニュー設定

- ウィンドウ: ボーダー付き
- ドキュメント: ボーダー付き、最大幅 40 文字
- アイテム表示: アイコン + 種別 + 省略表示 (最大 20 文字 + `…`)
- Copilot アイコン: ``

## その他設定

### インデント

| 設定 | 値 |
|------|-----|
| タブ幅 | 4 |
| シフト幅 | 4 |
| スペース展開 | 有効 |
| スマートインデント | 有効 |
| 折り返しインデント | 有効 |

### 検索

| 設定 | 値 |
|------|-----|
| インクリメンタル検索 | 有効 |
| 検索ハイライト | 有効 |
| 大文字小文字無視 | 有効 |
| スマートケース | 有効 (大文字入力時は区別) |

### ファイル保存・バックアップ

| 設定 | 値 |
|------|-----|
| バックアップ | 無効 |
| スワップファイル | 無効 |
| 永続アンドゥ | 有効 (`~/.config/nvim/undo/`) |

### 表示

| 設定 | 値 |
|------|-----|
| 行番号 | 有効 |
| 相対行番号 | 有効 |
| カーソルライン | 有効 |
| サインカラム | 常時表示 |
| True Color | 有効 |
| グローバルステータスライン | 有効 (laststatus=3) |
| タブライン | 常時表示 |
| 補完メニュー最大行 | 12 |
| スクロールオフセット | 4 行 |

### オートコマンド

| イベント | 動作 |
|---------|------|
| ターミナルオープン | 自動で Insert モードに入る、行番号・サインカラムを非表示 |
| help / man / qf 等 | サインカラムを非表示 |
| 外部ファイル変更 | 自動でバッファを再読み込み |
| ファイルを開く | 前回のカーソル位置を復元 |

### ステータスライン (lualine)

モードに応じて色が変化し、以下を表示:
- 現在のモード
- ファイルサイズ・ファイル名・カーソル位置
- 診断情報 (エラー / 警告 / 情報)
- アクティブな LSP サーバー名
- ファイルエンコーディング・フォーマット
- Git ブランチ名・差分統計 (追加 / 変更 / 削除行数)

### 無効化されたデフォルトプラグイン

パフォーマンス向上のため以下を無効化:

- `gzip`
- `matchit`
- `matchparen`
- `netrwPlugin`
- `tarPlugin`
- `tohtml`
- `tutor`
- `zipPlugin`
